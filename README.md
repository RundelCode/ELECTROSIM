# ELECTROSIM

ELECTROSIM es una aplicación de escritorio desarrollada en JavaFX que permite simular y analizar interacciones electrostáticas entre cargas eléctricas mediante la Ley de Coulomb.

El sistema permite:

- Crear y configurar múltiples cargas eléctricas positivas y negativas.
- Visualizar su comportamiento sobre un plano cartesiano interactivo.
- Calcular fuerzas eléctricas entre cargas en tiempo real.
- Analizar magnitudes físicas como campo eléctrico, potencial eléctrico y trabajo mediante gráficas dinámicas.
- Consultar resultados individuales y globales del sistema.
- Registrar y almacenar simulaciones junto con sus gráficas, configuraciones y observaciones en una bitácora experimental.

El proyecto fue desarrollado bajo arquitectura MVC, combinando simulación física, visualización de datos y registro experimental en una interfaz interactiva.
